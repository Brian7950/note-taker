## Description: <a name="description"></a>
### Note Taker is built ontop of Nodejs for anyone to use an application to be able to Read, Write, and Delete notes. The application uses express to access backend to be able to save notes and retrieve previous note data. 

## Installation: <a name="installation"></a>
### on command line use commmand: npm install 
### this will install needed dependencies to be able to run application. 

## Technologies <a name="technologies"></a>
### HTML, CSS, JavaScript, Nodejs, express module

## Usage: <a name="usage"></a>
### Note taker is functional through expresss to be able to retrieve all information. Application is deployed through heroku that way you can shut off machine and not worry server will stop responding. Application gives user ability to create notes. Gives user a note title and note details. Gives user ability to add aditional notes, save and delete if desire.
